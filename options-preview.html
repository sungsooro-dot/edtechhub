장점

  • 홈이 가장 단순해짐 — 방문자가 빠르게 원하는 곳으로 이동

장점

  • 현재 코드 기반 — 대형 재구축 없이 바로 적용 가능
  • 모든 이니셔티브가 홈에 보임 (카드 1줄로 정리)
  • EdTech Together가 명확한 메인 CTA로 부각됨
  • 중복 콘텐츠 제거만으로 섹션 14개 → 9개
  • Django 전환 전에도 바로 개선 효과 체감

단점

  • 여전히 스크롤이 존재 (Option 1보다 긺)
  • 세부 이니셔티브 설명이 카드 1줄에만 국한됨
  • 나중에 서브 페이지로 확장 필요 (Option 1과 병행)
작업 난이도
